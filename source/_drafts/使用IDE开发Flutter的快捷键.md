# Alt + Enter/Option+Enter

1. 选中 StatelessWidget，使用如上快捷键，将 StatelessWidget 转换为 StatefulWidget。

2. 选中 Widget 名字，使用以上快捷键，可以为指定 Widget 添加相应的 Widget。
