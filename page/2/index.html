Kotlin 核心编程(五)：面向对象

0x0001 Kotlin 中的类

Kotlin 中类与 Java 中的几点不同:

  • 在 Kotlin 中除非显示声明延时初始化，那么属性需要显式的指定默认值。
  • val 为不可变属性
  • 修饰符的访问权限不同，Kotlin 默认全局可见， Java 默认包可见。
